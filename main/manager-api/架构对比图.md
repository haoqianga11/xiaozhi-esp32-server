# Spring Boot vs Android MVVM 架构对比

## 标准分层对比

```text
Android MVVM                     Spring Boot 分层架构
┌─────────────────┐              ┌─────────────────┐
│   View (UI)     │ HTTP请求 ←→  │   Controller    │ 接收请求/返回响应
│  - Activity     │              │  - @RestController │
│  - Fragment     │              │  - @GetMapping  │
│  - Compose      │              │  - @PostMapping │
└─────────────────┘              └─────────────────┘
         ↕ 数据绑定                        ↕ 方法调用
┌─────────────────┐              ┌─────────────────┐
│   ViewModel     │    业务逻辑  │    Service      │ 业务逻辑处理
│  - LiveData     │      ←→      │  - @Service     │
│  - StateFlow    │              │  - @Transactional │
│  - 状态管理      │              │  - 业务规则      │
└─────────────────┘              └─────────────────┘
         ↕ 调用                           ↕ 调用
┌─────────────────┐              ┌─────────────────┐
│   UseCase       │   业务封装   │   Service       │ 业务封装
│  - 单一职责      │      ←→      │  - 复杂业务拆分  │
│  - 业务用例      │              │  - 事务管理      │
└─────────────────┘              └─────────────────┘
         ↕ 调用                           ↕ 调用  
┌─────────────────┐              ┌─────────────────┐
│   Repository    │   数据抽象   │     DAO         │ 数据访问
│  - 数据源抽象    │      ←→      │  - @Repository  │
│  - 本地/远程     │              │  - MyBatis      │
└─────────────────┘              └─────────────────┘
         ↕ 调用                           ↕ SQL执行
┌─────────────────┐              ┌─────────────────┐
│   DataSource    │   具体数据   │   Database      │ 数据存储
│  - Room DAO     │      ←→      │  - MySQL        │
│  - Retrofit API │              │  - Redis        │
│  - DataStore    │              │  - 文件系统      │
└─────────────────┘              └─────────────────┘
```

## 数据流转对比

```text
Android MVVM 数据流:               Spring Boot 数据流:
UI事件 → ViewModel                HTTP请求 → Controller
   ↓                                ↓
ViewModel → UseCase               Controller → Service  
   ↓                                ↓
UseCase → Repository              Service → DAO
   ↓                                ↓  
Repository → DataSource          DAO → Database
   ↓                                ↓
DataSource → 数据变更             Database → 数据变更
   ↓                                ↓
LiveData/Flow → UI更新            Response → HTTP响应
```
